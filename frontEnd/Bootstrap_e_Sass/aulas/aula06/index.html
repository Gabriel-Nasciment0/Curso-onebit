Conhecendo o Sistema de Grid do Bootstrap

Baseado em Flexbox

Redimensiona automaticamente qualquer número de colunas, trabalha forma responsiva e suporta a maioria das classes utilitárias do Flexbox.

Organizado para 12 Colunas

Permite definir tamanhos específicos para as colunas se baseando em um sistema de grade com 12 colunas.

Permite Uso de Breakpoints

Possui classes modificadoras que ativam a distribuição em colunas apenas a partir de um breakpoint, garantindo uma melhor responsividade.
